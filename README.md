# Tall Man Training

Mobile-friendly web app for daily strength training based on the **Tall Man Training** protocol by Alexander Juan Antonio Cortes.

## Features

- **5-day hypertrophy split** (Mon-Fri) with rest days (Sat/Sun)
- **Exercise cards** with detailed form cues, tempo prescriptions, and key pointers
- **In-workout logging** - track weight, sets, reps, and notes per exercise
- **Downloadable workout report** (plain text)
- **Rest timer** - floating stopwatch for rest periods
- **Week selector** (Weeks 1-12)
- **Dark/light theme** toggle
- **Bottom navigation** - Today / Week / Log views
- **PWA-ready** - add to home screen on iPhone for a native app feel

## Training Split

| Day | Focus |
|-----------|--------------------------------------|
| Monday | Shoulders + Secondary Chest |
| Tuesday | Arms |
| Wednesday | Lower Body - Quads / Glutes / Hams |
| Thursday | Back + Biceps |
| Friday | Chest / Shoulders / Triceps |
| Sat-Sun | Rest |

## Usage

Open `index.html` in a browser, or deploy to any static hosting provider.

On iPhone: Open in Safari > Share > Add to Home Screen.

## Tech Stack

Vanilla HTML, CSS, and JavaScript - no frameworks, no build step.
